# Auth Pages - Images Guide

## Required Images

### 1. **just.jpg**
- **Purpose**: Background image for Login and Signup pages
- **Used by**: 
  - `Login.jsx`
  - `Signup.jsx`
- **Recommended Size**: 1920x1080px or larger
- **Format**: JPG/JPEG
- **Description**: Should be a professional parking-related background image

### 2. **car.webp**
- **Purpose**: Background image for Landing page
- **Used by**: 
  - `Landing.jsx`
- **Recommended Size**: 1920x1080px or larger
- **Format**: WebP (for better performance)
- **Description**: Should showcase parking/car theme

## Setup Instructions

1. **Download or create your images** that match the parking management theme
2. **Place them in this folder** (`auth-pages/assets/`)
3. **Ensure the filenames match exactly**:
   - `just.jpg`
   - `car.webp`
4. The components will automatically reference these images

## Image Suggestions

- **just.jpg**: Modern parking lot, parking structure, or parking space
- **car.webp**: Car in parking lot, parking facility, or parking sign

## Notes

- Images will improve the visual appeal of the authentication pages
- Use high-quality images for better user experience
- Optimize images for web (compress file sizes)
- Ensure images are at least 1920x1080 for responsive display
